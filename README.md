# Mise — Privacy Policy

This repository hosts the public privacy policy for the **Mise** pantry management app, served via GitHub Pages.

## Live URL

**https://dacraven95.github.io/mise-privacy-policy/**

This URL is linked from the App Store and Google Play listings.

## Making changes

Edit `index.html` directly — the page will update automatically within a few minutes of pushing.

Remember to update the **"Last updated"** date near the top of the file whenever the policy content changes.

## Enabling GitHub Pages

1. Go to **Settings → Pages** in this repo
2. Under **Source**, select **Deploy from a branch**
3. Set branch to `main` and folder to `/ (root)`
4. Save — the site will be live at the URL above within a minute or two
